1- Description

Python code to generate images of the 2D game Break Bricks
this is NOT an interactive user game, it only generate images, for the purpose of creating videos, the game plays by itself
Developed and tested on Macos only

Licence : CC0

2- dependencies: 

uses the python packages:
os
random
colorsys
sys
pygame

3- Installation

Unzip the file in any folder

4- Execution

to run the code

cd [folder where the code is]
python3 ./draw-bricks-07.py

5- Outputs

In the folder ./output_images 
there are the *.bmp images files
that can be added in a video editing software like blender
as an Image/Sequence

The file collision.txt
contains the frames and collision type of the ball against walls or bricks
so that we can add automatically the sound effects in blender video editing
